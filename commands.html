Linux Command API - Commands

Here are the details of available Linux commands in the API:

Available Commands

GET /commands

This endpoint retrieves a list of all available Linux commands.

Response Example:

[
    { "name": "ls", "description": "List directory contents", "example": "ls -l" },
    { "name": "tail", "description": "Show the last lines of a file", "example": "tail -n 10 file.txt" }
]

GET /commands/:name

This endpoint retrieves details about a specific command.

Example Request:

GET /commands/ls

Example Response:

{
    "name": "ls",
    "description": "List directory contents",
    "example": "ls -l"
}

POST /commands

This endpoint adds a new Linux command to the API.

Example Request:

{
    "name": "grep",
    "description": "Search text using patterns",
    "example": "grep -r 'pattern' folder/"
}

Example Response:

{
    "message": "Command added successfully"
}
